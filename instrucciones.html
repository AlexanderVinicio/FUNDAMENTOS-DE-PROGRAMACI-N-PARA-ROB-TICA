Para realizar las actividades de este día, utilizaremos PSeint, una herramienta que permite escribir, probar y depurar pseudocódigos de manera sencilla. Si no lo tienes instalado, sigue las siguientes instrucciones.

Enlaces de descarga:

https://pseint.sourceforge.net/?page=descargas.php

1. Descárgalo, de acuerdo a tu sistema operativo.

2. Sigue las instrucciones en pantalla.

3. Abre PSeint.
