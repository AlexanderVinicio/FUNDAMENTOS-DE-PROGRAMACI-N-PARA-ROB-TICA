Contenido 2: Sentencias Condicionales (si – entonces, mientras – repetir hasta)

Fundamentación Teorica

Sentencias condicionales (si – hacer mientras – repetir hasta)

Algunos de las sentencias utilizadas son:

Si: Evalúa una condición y ejecuta un bloque de código si dicha condición es verdadera.
Hacer mientras:  Repite un conjunto de instrucciones mientras una condición se cumpla, en cuanto la condición no se cumple el ciclo deja de ejecutarse.
Repetir hasta: Ejecuta una secuencia de instrucciones hasta que la condición sea verdadera.
En este sentido estas estructuras son esenciales para desarrollar algoritmos de aprendizaje automático y toma de decisiones, permitiendo a los robots aprender de su entorno y adaptarse a nuevas situaciones así mismo, permite que los robots puedan interactuar con su entorno de manera flexible y adaptativa, llevando a cabo tareas complejas y diversas.

Si - Entonces


Imaginemos que están a punto de salir de casa para dirigirte a la escuela, y no sabes si hace frío o calor. Para tomar una decisión, se utiliza una estructura condicional llamada: Sí-Entonces para que el proceso sea más fácil.


La condición es: Mirar por la ventana o revisar en el celular el pronóstico del clima.

Y si la temperatura es menor a 15 grados, entonces debes de llevar un abrigo.

Y si la temperatura es mayor a 25 grados, entonces debes utilizar ropa ligera.


En pseudocódigo:


Algoritmo ejemplo_si_entonces

Definir temperatura Como Real

Escribir "¿Cuál es la temperatura en grados Celsius?"

Leer temperatura

Si temperatura < 15 Entonces

Escribir "Hace frío, ponte un abrigo."

Sino

Si temperatura > 25 Entonces

Escribir "Hace calor, ponte ropa ligera."

Sino

Escribir "El clima está templado, ponte algo cómodo."

FinSi

FinSi

FinAlgoritmo

Ejecución:

Mientras

El programa pedirá al usuario que ingrese un número positivo. Si el usuario ingresa un número negativo, el programa le pedirá otro número hasta que ingrese uno positivo.


Pseudocódigo:


Algoritmo ejemplo_mientras

    Definir numero Como Entero

    Escribir "Ingresa un número positivo:"

    Leer numero

    Mientras numero < 0 Hacer

        Escribir "El número no es positivo. Intenta de nuevo:"

        Leer numero

    FinMientras

    Escribir "Has ingresado un número positivo:", numero

FinAlgoritmo


Ejecución:

Como podemos observar la función del mientras, repite una acción hasta que se cumpla una condición.

Repetir - Hasta

En este ejemplo se pedirá al usuario un número y lo continuará pidiendo hasta que sea mayor a 10.


Pseudocódigo:

Algoritmo ejemplo_repetir

    Definir numero Como Entero

    Repetir

        Escribir "Ingresa un número mayor a 10:"

        Leer numero

    Hasta Que numero > 10

    Escribir "Has ingresado un número mayor a 10:", numero

FinAlgoritmo


Ejecución:

El repetir es útil cuando queremos ejecutar al menos una vez y luego repetirlo hasta cumplir una condición.
